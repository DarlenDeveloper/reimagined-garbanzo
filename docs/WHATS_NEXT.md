## üìã BRAND RULES CHECKLIST

Use this for each screen you update:

- [ ] All text uses Poppins font
- [ ] Main red (#fb2a0a) for primary elements (follow buttons, verification badges, active states)
- [ ] Button red (#b71000) for CTA buttons
- [ ] Border radius = height / 2 for ALL elements
- [ ] NO gradients
- [ ] NO borders on text inputs (use InputBorder.none)
- [ ] NO shadows (unless explicitly requested)
- [ ] NO sharp corners anywhere
- [ ] Verification badges use main red (#fb2a0a)
- [ ] Logo is square with 24px border radius (NOT circular)
